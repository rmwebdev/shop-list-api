This is the version 1 of [Daftar Belanja](https://daftar-belanja.herokuapp.com/). More cleaned and user friendly code are applied.

## 📐 Project setup
```
npm install
```

### 🚀 Compiles and hot-reloads for development
```
npm start
```

### Compiles and minifies for production
```
npm run build
```
